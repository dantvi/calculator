# Calculator Improvements

This file documents enhancements made to the original Calculator project. The improvements focus on usability, functionality, and visual design, aiming to make the calculator more user-friendly and modern.

## Script improvements:
- Displays history
- Allows exponentiation and square roots
- Allows toggling sign between positive and negative
- Enhances result formatting by limiting the display to 14 digits
- Converts large numbers to scientific notation
- Allows typing on keyboard
- Allows backspace to delete last digit

## Style improvements:
- Made the style feel more modern with a carbon color scheme
- Uses proportional aspect ratio of 2/3
- Improved the overall look of the calculator to be more visually appealing
